# tailwind_responsive_page

## ▶️ How to Run the Project

You can run this project using either of the following methods:

### Option 1: Open the HTML File

1. Download or clone this repository to your computer.
2. Open the project folder.
3. Locate the `index.html` file.
4. Double-click the file to open it in your web browser.

### Option 2: Run with VS Code (Recommended)

1. Open the project folder in **Visual Studio Code**.
2. Install the **Live Server** extension if you haven't already.
3. Right-click on the `index.html` file.
4. Select **Open with Live Server**.
5. Your default browser will open the project automatically, and any changes you save will be reflected after refreshing (or automatically if Live Server is configured to do so).
